# agentsio.dev — site starter

Single-file landing page. Deploy the whole folder to any static host
(Netlify, Vercel, Cloudflare Pages, GitHub Pages). `index.html` is the page;
favicons + og-card.png sit beside it; `assets/logo/` holds the full logo kit.

## Edit points (search "EDIT ME" in index.html)
- <title> and <meta name="description"> — your search/social text
- Booking links — currently mailto:hello@agentsio.dev; swap for Cal.com/Calendly
- og:image / og:url — set to your live https://agentsio.dev once deployed
- Footer details (email, location, year)
